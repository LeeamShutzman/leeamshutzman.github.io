Cool Guy App

Github Repository   

What is it?

I created this app because I wanted a way to keep a streak of how many days in a row I accomplished what I wanted to get done. I decided to do this as a way to keep myself accountable for achieving my daily goals.

How does it work?

When the app is first downloaded, the user is prompted to enter how many days they've been "cool," up until that point, however they wish to define cool. On the same page, the user chooses an hour of the day to be sent a daily notification. The notification asks if the user has been cool that day and if they say "no," the counter resets to zero. If they say "yes," the counter increments by one. The user can either answer the prompt by going to the app directly from their homescreen, clicking on the notification, or clicking the action buttons on the notification. The user can also click the settings button to go to the setup screen if they would like to manually change the counter or if they would like to change the time of the notification.
